Feature Request 1: Sponsored / Featured ranking for Clubs and Tournaments

I want to add a Sponsored / Featured ranking system for both clubs and tournaments/events so that super admins can manually override the normal sorting order and prioritize paid placements.

Goal

Normally, clubs and tournaments should sort the usual way. But when something is marked as sponsored, I want it to be able to rise above the normal sort and be featured higher in the list.

Admin behavior

This setting should be available only in:
	•	Club Settings for clubs
	•	Tournament Settings for tournaments/events

It should be visible only to super admins.

Fields

Add the following fields to both clubs and tournaments:
	•	Sponsored: checkbox / toggle
	•	Sponsored Rank: numeric ranking from 1 to 10

Rules
	•	If Sponsored is not checked, the club or tournament follows the normal sorting logic.
	•	If Sponsored is checked, the item can override normal sorting and be prioritized.
	•	Higher sponsored rank should appear higher in the list.
	•	10 is the highest priority and should always appear above 9, 8, etc.
	•	Sponsored items should appear ahead of non sponsored items.
	•	If multiple sponsored items have the same sponsored rank, then fall back to the normal secondary sort.

Display behavior for clubs

If a club is marked as sponsored / featured, show a small visual badge on the club card or listing:
	•	a small star icon
	•	text next to it that says: Featured Club

This should be subtle, clean, and easy to notice.

Display behavior for tournaments

Do the same type of system for tournaments or events.
If a tournament is sponsored / featured, it should also:
	•	be prioritized in list sorting
	•	optionally show a small badge such as Featured Event or Featured Tournament

Suggested implementation notes
	•	Keep the existing default sorting as the fallback
	•	Apply sponsored sorting first, then normal sorting after that
	•	Restrict editing of sponsored fields to super admins only
	•	Add the fields to the database, API, admin forms, and public list queries

Feature Request 2: Match Stats section on the Match Page

I want to add a Match Stats section on the match page, displayed above the match timeline.

Goal

The match page should show a quick statistical summary for that specific game, with one row per team, so viewers can compare team performance at a glance.

Layout

Place a Match Stats table above the match timeline.

Use two rows:
	•	one row for Team A
	•	one row for Team B

Stats to track and display

For each team, track and display:
	•	Goals
	•	Fouls
	•	Bowls In
	•	Knock Ins

These stats should be tied to that specific match only.

Additional stat suggestions

Here are two good additional technical stats that fit naturally with the data you are already collecting or could collect easily:
	•	Penalty Goals
	•	Shots on Goal

If you want alternatives, two other good options would be:
	•	Timeouts Used
	•	Yellow / Red Cards

But my top recommendation would be:
	•	Penalty Goals
	•	Shots on Goal

because they make the match page feel more complete and analytical.

Rules / behavior
	•	Stats should update based on match events or admin input for that game
	•	Stats should belong to the specific match, not season wide totals
	•	Stats should display clearly in a side by side comparison format
	•	This section should appear before the timeline so users see the summary first

UX intent

The match page should feel more professional and complete by giving:
	•	a quick team comparison at the top
	•	then the detailed event by event timeline below
