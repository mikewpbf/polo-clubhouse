MVP SYSTEM

On the tournament page admin side, allow the admin to select the MVP for that tournament.

MVP selection flow:
- Choose team
- Choose player from that team

Once selected, the MVP section should be able to show:
- Player Name
- Team Name
- Games Played in Tournament
- Goals Scored in Tournament

This should pull from tournament and match data when possible.

We should also allow an admin override in case the admin wants to manually adjust the displayed numbers or text for a specific tournament.

BEST PLAYING PONY SYSTEM

On the tournament page admin side, allow the admin to select the Best Playing Pony for that tournament.

Best Playing Pony selection flow:
- Choose team
- Choose player from that team
- Choose horse from that player’s saved horse list

Important:
The horse should come from the saved horse list under that player in Team Settings, so we are not manually retyping horse details every time.

Please structure the data model cleanly for future AI workflows.
